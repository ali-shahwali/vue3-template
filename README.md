# Vue 3 Vite Template

This template includes

 - Vue 3
 - Vite
 - TypeScript support
 - File based routing
 - WindiCSS
 - Auto importing components
 - Iconify

## Launch app

### Install dependencies
```bash
$ npm install
```
or
```bash
$ yarn
```
or
```bash
$ pnpm install
```
#### Development server
*With npm*

```bash
$ npm run dev
```
*With yarn*
```bash
$ yarn dev
```
*With pnpm*
```bash
$ pnpm dev
```
